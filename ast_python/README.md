# Meet Translator - 实时同声传译工具

Meet Translator 是一款基于豆包同声传译2.0 API 的实时语音翻译工具，可以在会议、通话等场景中实时将语音翻译成目标语言。

---

## 功能特性

- **实时翻译**：低延迟端到端语音翻译
- **多语言支持**：支持中文、英文、日文、韩文等多种语言的互译
- **虚拟声卡**：通过 BlackHole 虚拟声卡实现音频路由
- **许可证管理**：支持体验卡、天卡、周卡、月卡、年卡等多种授权方式

---

## 系统要求

- macOS 12.0 或更高版本
- 已安装 BlackHole 虚拟声卡
- Python 3.8+（如需源码运行）

---

## 第一步：申请豆包同声传译2.0 API Key

在开始使用之前，您需要申请豆包同声传译2.0 API Key：

**申请地址**：[豆包同声传译2.0 API 申请](https://www.volcengine.com/docs/6561/1756902?lang=zh)

申请步骤：

1. 注册并登录火山引擎账号
2. 开通同声传译2.0 服务
3. 创建应用并获取 App Key 和 Access Key
4. 将获取的密钥配置到软件设置中

---

## 第二步：安装 BlackHole 虚拟声卡

BlackHole 是一款 macOS 虚拟声卡，用于将系统音频路由到 Meet Translator 进行翻译。

### 安装步骤

**方法一：使用 Homebrew（推荐）**

```bash
brew install blackhole-2ch
```

**方法二：手动下载安装**

1. 访问 BlackHole 官网：https://existential.audio/blackhole/
2. 下载 BlackHole 2ch 版本
3. 打开下载的 `.pkg` 安装包，按提示完成安装
4. 安装后需要在「系统偏好设置」→「安全性与隐私」→「隐私」中允许「麦克风」权限

### 验证安装

安装完成后，打开「活动监视器」或「音频 MIDI 设置」，确认 BlackHole 2ch 已显示在音频设备列表中。

---

## 第三步：配置会议软件

### 设置系统音频输出

1. 打开「系统偏好设置」→「声音」
2. 切换到「输出」标签
3. 选择「BlackHole 2ch」作为输出设备

或者使用快捷方式：

1. 点击菜单栏的音量图标
2. 选择「选择音频输出设备」
3. 选择「BlackHole 2ch」

### 设置会议软件

以 Zoom 为例：

1. 打开 Zoom 设置
2. 进入「音频」设置
3. 在「麦克风」下拉菜单中选择「BlackHole 2ch」
4. 保存设置

其他会议软件（腾讯会议、Teams、钉钉等）设置方法类似，在音频设置中找到「BlackHole 2ch」并选为输入设备。

---

## 第四步：配置 API Key

首次使用软件时：

1. 打开 Meet Translator 应用
2. 点击界面上的「设置」按钮
3. 在 API Key 输入框中粘贴您申请的 App Key
4. 点击保存

---

## 第五步：生成并激活密钥

### 命令行生成密钥

在终端中运行以下命令生成密钥：

```bash
cd /path/to/ast_python
python3 license_manager.py --type trial        # 生成体验卡（1小时）
python3 license_manager.py --type day          # 生成天卡（1天）
python3 license_manager.py --type week         # 生成周卡（7天）
python3 license_manager.py --type month        # 生成月卡（30天）
python3 license_manager.py --type year         # 生成年卡（365天）
python3 license_manager.py --type trial -c 10  # 批量生成10张体验卡
```

### 激活密钥

1. 在软件界面中点击「设置」→「许可证」
2. 在许可证输入框中粘贴生成的密钥
3. 点击「激活」按钮
4. 激活成功后即可开始使用

## 常见问题

### Q1：为什么必须戴耳机？

A：为了避免声音循环回授。如果不戴耳机，会议软件会同时收录翻译后的声音，形成循环，影响翻译效果。

### Q2：BlackHole 2ch 找不到？

A：请确认已正确安装 BlackHole，如果仍然找不到，尝试重启电脑。

### Q3：会议软件听不到声音？

A：这是正常的。音频已路由到 BlackHole 进行翻译。如果您需要听到对方的声音，可以在「系统偏好设置」→「声音」→「输出」中同时选择「MacBook Pro 扬声器」和「BlackHole 2ch」（使用「应用程序」音量设置）。

### Q4：API Key 配置失败？

A：请确认您使用的是豆包同声传译2.0 的 App Key，而非其他服务的密钥。

### Q5：密钥激活失败？

A：请检查：

- 网络连接是否正常
- 密钥是否已过期
- 是否已在其他设备激活（密钥已绑定设备）
